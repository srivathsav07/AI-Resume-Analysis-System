## How to Use Test Data

1. Copy the "Sample Resume Text" into a text editor
2. Save as `sample_resume.txt`
3. Paste the "Sample Job Description" into the app's JD field
4. Run the analysis to see expected outputs

This will help you understand how the system works before uploading your actual resume.
